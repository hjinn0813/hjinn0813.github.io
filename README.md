# Make web portfolio by github.io

개인의 기술 스택과 작업 내역 정리를 위해 기획 및 제작한 포트폴리오 사이트입니다.<br/>

## 💻 주요 기능

- `About`에서 텍스트 타이핑 애니메이션

- 스크롤하면 Header가 자연스럽게 사라지는 기능 (GSAP)

- 내비게이션 바 hover시 언더라인 이벤트

- `프로젝트` 영역의 카드 플립 애니메이션

- 화면 우측 하단의 '탑 버튼' 기능

- 반응형 미디어쿼리 적용
